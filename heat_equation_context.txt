The forward problem: given alpha, solve dT/dt = alpha * d²T/dx²

Discretization:
    T_new[i] = T[i] + r * (T[i+1] - 2*T[i] + T[i-1])
    where r = alpha * dt / dx²

Grid: x = 0 to 1, dx=0.01, dt auto-set, t_final as given
Initial condition: T(x,0) = exp(-(x-0.5)² / (2*0.1²))
Boundary conditions: T=0 at x=0 and x=1

Large alpha = faster diffusion = flatter profile.
Small alpha = slower diffusion = taller peak.

To evaluate your alpha call compare_to_target(alpha).
It returns MSE and gradient dMSE/dalpha.
Update rule: alpha = alpha - (0.1 * alpha / abs(gradient)) * gradient
Start with alpha=0.01. Stop when MSE < 1e-5. Alpha is between 0.001 and 0.02.
You MUST call compare_to_target to get MSE — do NOT compute it yourself.
